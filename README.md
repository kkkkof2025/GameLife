# 💰 LifeCash - 人生财富沙盘模拟器

> 体验一场人生，学会财富智慧

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform](https://img.shields.io/badge/platform-Web%20Pages-blue)](https://pages.github.com/)

## 🎮 游戏简介

LifeCash 是一款金融沙盘模拟游戏，让你在虚拟世界中体验完整的人生旅程。从选择职业开始，通过工作、投资、社交等决策，管理你的健康、幸福和财富，最终实现财务自由或成为人生赢家。

## 🚀 快速开始

### 在线游玩

部署到 GitHub Pages 后访问：
```
https://yourusername.github.io/lifecash/
```

### 本地运行

1. 下载或克隆本项目
2. 直接用浏览器打开 `index.html` 文件
3. 选择职业，开始游戏

```powershell
# Windows 用户
start index.html

# macOS 用户
open index.html

# Linux 用户
xdg-open index.html
```

## 🎯 游戏功能

### 游戏模式

| 模式 | 说明 |
|------|------|
| 🎮 畅玩模式 | 无时间限制，自由探索人生，破产后可东山再起 |
| ⏱️ 限时模式 | 30分钟/1小时/2小时限时挑战，输出最终报告 |
| 👥 多人模式 | 与好友同局竞技，实时排行（纯前端实现） |

### 职业系统

| 职业 | 启动资金 | 月薪 | 特点 |
|------|---------|------|------|
| 👔 工薪族 | ¥10,000 | ¥5,000 | 稳定收入 |
| 💻 自由职业者 | ¥5,000 | ¥3,000 | 灵活接单 |
| 🚀 创业者 | ¥50,000 | ¥0 | 高风险高回报 |
| 🏛️ 公务员 | ¥8,000 | ¥4,000 | 人脉广泛 |
| 📊 投资人 | ¥100,000 | ¥0 | 财务分析 |
| 📚 教师 | ¥6,000 | ¥4,500 | 教育加成 |

### 投资系统

| 投资品种 | 年化收益 | 风险等级 | 流动性 |
|---------|---------|---------|--------|
| 🏦 银行存款 | 3% | ⭐ | 100% |
| 📜 国债 | 4% | ⭐⭐ | 95% |
| 📈 基金 | 15% | ⭐⭐⭐ | 80% |
| 💹 股票 | 25% | ⭐⭐⭐⭐⭐ | 90% |
| 🏠 房产 | 8% | ⭐⭐ | 30% |
| 🥇 黄金 | 10% | ⭐⭐⭐ | 70% |
| 🚀 创业投资 | 50% | ⭐⭐⭐⭐⭐ | 10% |

### 属性系统

- **❤️ 健康度** - 影响游戏状态，归零则游戏结束
- **😊 幸福度** - 决定人生质量
- **💰 金钱** - 核心资源
- **📊 等级** - 通过工作和学习提升

### 事件系统

游戏包含 12+ 种随机事件，分为：
- 🎯 **机会类** - 股市内幕、升职加薪、投资机会
- ⚠️ **风险类** - 突发疾病、裁员危机
- 👥 **社交类** - 婚礼聚会、贵人相助
- 🎲 **随机类** - 彩票中奖、交通违章

在 `docs/` 目录添加 `CNAME` 文件：

